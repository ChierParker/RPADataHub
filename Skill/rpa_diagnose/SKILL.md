# rpa-diagnose — 智能诊断

## 定位
输入异常问题，自动聚合上下文（历史同类异常、今日任务状态），AI 分析根因、方案、影响、置信度。

## 触发方式
- Admin → 智能运维 → 选择异常类型 + 描述问题 → 点击"诊断"
- Agent 对话: "诊断登录失败"

## 数据来源
- `rpa_exception_log` — 近 7 天同类异常数
- `task_queue` — 今日任务执行状态
- `alert_knowledge_base` — 历史解决方案

## 输出
```json
{
  "root_cause": "目标系统响应变慢或网络波动导致超时阈值触发",
  "suggestion": "1.检查网络延迟 2.增加超时阈值 3.分时段重试",
  "impact": "影响采集任务成功率，部分店铺数据缺失",
  "confidence": 85
}
```
